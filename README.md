# Markdown TOOL!

**Vue version 3.7.0**

## Vue-cli install

```
$ yarn global add @vue/cli
```

## Install packages

```
$ yarn add marked
$ yarn add highlightjs
```

## Run!

```
$ yarn run serve
```

http://localhost:8080
